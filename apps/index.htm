Getting Started

Integromat has evolved into Make

While Make has a ton of new features, we did our best to keep the process of creating custom apps the same as before.
We are working on updating this documentation to properly reflect our new brand identity, but you can use it even now to create your own Make apps.

Important Notes

  • All modules can be tested directly in scenarios.
  • Changes to communication configs are immediately active.
  • You can see raw requests/responses in your browser's console.
  • Changes in parameters and interface requires you to reload scenario editor page.
  • We use JSONC (JSON with comments) in all sections except common data.
